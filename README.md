# funjcam-ios

## Goals
* Testing GoogleCustomSearch
* Testing fastlane
* Testing NimbusKit

## Requirements
* iOS 9.0+
* xcode 7.3+
* CocoaPods 1.0.1

## Installation
```
[sudo] gem install cocoapods
pod setup
```

``` git clone https://github.com/BoxJeon/funjcam-ios
cd funjcam-ios
pod install
```
* open funjcam.xcworkspace
